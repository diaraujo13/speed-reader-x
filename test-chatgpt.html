# RSVP — leitura veloz

RSVP (**Rapid Serial Visual Presentation**) apresenta uma palavra de cada vez no mesmo ponto focal, eliminando *micro-sacadas* e regressões. Isso permite leitura significativamente mais rápida sem perda de compreensão para a maioria dos textos.
